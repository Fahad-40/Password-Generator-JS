# Password Generator

A **simple and interactive password generator** built using **HTML, CSS, and JavaScript**. Generate strong passwords quickly with customizable options for character types and length.

## Features

- Generate secure passwords with lowercase, uppercase, numbers, and special characters.
- Adjustable password length (4-24 characters) using a slider.
- Toggle password visibility with the eye icon.
- Copy password to clipboard with a single click.
- Stylish and responsive design using CSS and Google Fonts.

## Technologies Used

- **HTML5** for structure
- **CSS3** for styling
- **JavaScript** for functionality
- **Font Awesome** for icons
